CLI (Command Line Interface)

The CLI is a way to control a computer by typing commands instead of clicking with a mouse. It lets users talk directly to the system to run programs and manage files. Ethical hackers use it because it gives more control and works faster.

SSH (Secure Shell)

SSH is a tool that lets you safely connect to another computer over the internet. It protects your information by encrypting the connection so others cannot see it. Ethical hackers use SSH to manage systems securely.

Nmap

Nmap is a tool used to scan networks and find devices connected to them. It can show open ports and possible weak spots in a system. Ethical hackers use it to understand how secure a network is.

Wireshark

Wireshark is a tool that captures and shows data traveling across a network. It helps users see what information is being sent and received. Ethical hackers use it to find problems and suspicious activity.

Hydra

Hydra is a tool used to test login security by trying many username and password combinations. It helps show how strong or weak a login system is. Ethical hackers only use it with permission to improve security.

Brute Force Attack

A brute force attack is when someone tries many passwords until they find the correct one. This method takes time but can work if passwords are weak. Ethical hackers use this idea to show why strong passwords are important.
